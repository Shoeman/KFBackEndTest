# KFBackEndTest
Built using:
- NodeJS v18.x

## Setup

Checkout the project and run `npm install`

## Build Commands
_If using Windows: some commands will require Git Bash or PowerShell (Core)_

Clean: `npm run clean`

Compile:
`npm run compile`

Unit tests:
`npm run unittest`

Manual run (likely to change):
`npm run run-index`
